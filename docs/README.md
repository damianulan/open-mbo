# OpenMBO Documentation

OpenMBO is an employee performance and goal management platform built around MBO and OKR-style workflows.

This documentation is written for end users. It explains what the application is for, which areas are available in the interface, and what actions you can perform depending on your permissions.

## In this catalog

- [End User Guide](catalog/end-user-guide.md)

## What OpenMBO helps you do

- Follow ongoing performance campaigns.
- Review, complete, and discuss assigned objectives.
- Track your progress, statuses, deadlines, and reward points.
- View people in the organization and basic employment structure.
- Use notifications to stay informed about changes and assignments.
- Manage profile details and account preferences.
- If you have additional permissions, manage users, campaigns, templates, categories, and organizational settings.

## Important notes

- The interface is role-based. You only see modules and actions that your account is allowed to use.
- Some features are available only to coordinators, supervisors, HR, or administrators.
- The current application content is primarily prepared for Polish-speaking users.
